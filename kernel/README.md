# IphoxOS Kernel

The Kernel contains the core logic of IphoxOS.

It is not a collection of bot commands. It is the internal structure that allows IphoxOS to observe, understand, remember, feel and decide.

## Modules

- `BootMemory.cs` — remembers when IphoxOS was last started.
- `Pulse.cs` — represents the heartbeat of IphoxOS.
- `Brain.cs` — translates raw observations into meaning.
- `Memory.cs` — stores experience, not just data.
- `Heart.cs` — represents emotional state, energy, stress and comfort.
- `Director.cs` — will eventually decide what IphoxOS should do.
