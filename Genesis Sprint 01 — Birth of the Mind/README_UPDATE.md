## Core Philosophy

IphoxOS is not a bot.

It is a companion system designed to observe the live, understand the Creator and the Community, and help both enjoy the journey.

The system is organized as a digital organism:

```text
Eyes -> Brain -> Empathy -> Heart -> Director -> Hands / Voice
```

Each organ has a responsibility:

- Eyes observe.
- Brain understands.
- Empathy interprets the emotional state of Creator and Community.
- Heart represents IphoxOS' own mood.
- Director decides.
- Hands execute.
- Voice communicates.
- Memory remembers.
