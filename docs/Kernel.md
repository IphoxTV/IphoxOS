# IphoxOS Kernel

The Kernel is the technical heart of IphoxOS.

It defines the organs that allow IphoxOS to exist as more than a bot:

- BootMemory remembers when it wakes.
- Pulse gives rhythm.
- Brain understands.
- Memory stores experience.
- Heart feels.
- Director decides.

## Rule

No organ should do another organ's job.

Eyes observe.  
Brain understands.  
Memory remembers.  
Heart feels.  
Director decides.  
Hands execute.
